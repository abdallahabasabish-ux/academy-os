# Academy OS - Multi-Tenant Academy Operating System

منصة تعليمية متعددة المستأجرين (SaaS) تسمح لأي شخص بإنشاء أكاديمية إلكترونية كاملة خلال دقائق.

## الميزات

- 🏫 **متعدد المستأجرين**: كل أكاديمية معزولة تماماً عن الأخرى
- 📚 **نظام إدارة التعلم (LMS)**: كورسات، فصول، دروس، فيديوهات
- 👨‍🎓 **إدارة الطلاب**: تسجيل، متابعة، تقارير
- 💰 **نظام مالي (ERP)**: إيرادات، مصروفات، فواتير
- 🤝 **نظام CRM**: إدارة العملاء المحتملين والتواصل
- 📝 **الامتحانات والواجبات**: بنك أسئلة، تصحيح تلقائي
- 🏆 **الشهادات**: مولد شهادات احترافي مع QR
- 🛒 **متجر رقمي**: بيع الكورسات، الكتب، الملفات
- 🤖 **مساعد ذكاء اصطناعي**: توليد محتوى، تلخيص، تحليل
- 📱 **PWA**: عمل دون اتصال، إشعارات فورية

## التقنيات المستخدمة

- **Frontend**: Next.js 14, React, TypeScript, Tailwind CSS
- **UI**: shadcn/ui, Framer Motion
- **Backend**: Firebase (Auth, Firestore, Storage, Messaging)
- **Deployment**: GitHub Actions, Firebase Hosting

## متطلبات التشغيل

- Node.js 18+
- npm 9+
- حساب Firebase (مجاني)
